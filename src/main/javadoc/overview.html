The Exploding Atoms application is an implementation of a known game for computers of the same name.

User Interface

The program uses the Java Swing library to provide a user interface. Components are created in the code manually without any configuration files.

Artificial Intelligence

The Minimax algorithm was used to find the best possible moves for A.I. player. Alpha-beta pruning allows to omit evaluation of some game states and speeds up computation.

Project Decomposition

The code is separated into two packages. One is responsible for the user interface and the other for the game logic and computation.
